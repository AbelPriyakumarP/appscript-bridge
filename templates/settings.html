This is your application's base URL. Use ngrok to expose it for external access.

API Usage Example

Send events to your webhook using any HTTP client:

curl -X POST {{ base_url }}/webhook/trigger \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "X-Event-Type: incident.created" \
  -d '{"message": "Hello from ServiceNow", "data": {}}'
